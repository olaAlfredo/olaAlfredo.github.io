Eliminar tarea

Elimina permanentemente una tarea del sistema. Este endpoint permite eliminar una tarea con todos sus detalles asociados.

Se elimina un objeto con el ID asignado y todos los datos de la tarea creada.

AUTORIZACIÓN REQUERIDA: ApiKey

Única por usuario. Permite crear, obtener y administrar tareas en un entorno de prueba para un usuario en específico.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

PARAMETROS

id string required
Id del objeto a eliminar.

Responses

201 La tarea ha sido eliminada
500 No se pudo eliminar la tarea
