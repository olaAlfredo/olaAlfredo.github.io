Modelo usuario

El modelo de usuario es una representación estructurada de la información relacionada con un usuario en un sistema. Este modelo se utiliza para gestionar la autenticación y la autorización, asegurando que los usuarios puedan acceder a sus cuentas de manera segura y eficiente.

AUTORIZACIÓN REQUERIDA: ApiKey

Única por usuario. Permite crear, obtener y administrar tareas en un entorno de prueba para un usuario en específico.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

REQUEST BODY SCHEMA: application/json

username string required
Representa el nombre de usuario.
password string required
Representa la contraseña del usuario.
json
[ { "username": "user 1", "password": "hashed_password_1" }, { "username": "user 2", "password": "hashed_password_2" } ]
