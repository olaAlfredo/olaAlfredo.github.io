Obtener tareas

Obtiene la lista completa de tareas disponibles en el sistema. Este endpoint permite obtener las tareas registradas con todos sus detalles asociados.

Se devolveran objetos con el ID asignado y todos los datos de las tareas creadas.

AUTORIZACIÓN REQUERIDA: ApiKey

Única por usuario. Permite crear, obtener y administrar tareas en un entorno de prueba para un usuario en específico.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

Responses

200 Obtuvo tareas encontradas
404 No se encontraron tareas
