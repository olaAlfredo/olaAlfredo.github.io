Editar tarea

Modifica una tarea existente. Este endpoint permite actualizar cualquier campo de la tarea.

Una vez seleccionado la tarea que se quiere modificar, se devolverá un objeto con el ID asignado y todos los datos de la tarea.

AUTORIZACIÓN REQUERIDA: ApiKey

Única por usuario. Permite crear, obtener y administrar tareas en un entorno de prueba para un usuario en específico.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

PARAMETROS

id string required
Id del objeto a actualizar.
REQUEST BODY SCHEMA: application/json
titulo string required
Representa el título o nombre de la tarea.
descripcion string required
Proporciona una descripción detallada de la tarea.

Responses

201 La tarea ha sido actualizada
500 No se pudo actualizar la tarea

Request samples

PUT
json
{ "titulo":"Estudiar para el examen de matemáticas", "descripcion":"Revisar los temas del capítulo 5 y resolver problemas del libro." }

Response samples

201
json
{ "id":"890ce6c56d04f820aa8438af", "titulo":"Estudiar para el examen de español", "descripcion":"Revisar los temas del capítulo 1 y 5.", "estado":"false" }

500
json
{ "code":"500", "message":"No se pudo actualizar la tarea" }
