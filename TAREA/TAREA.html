Modelo tarea

El modelo de tarea es una representación de los datos relacionados con las tareas de un usuario en el sistema. Este modelo define los atributos principales de una tarea, como su título, descripción y estado, permitiendo su creación y almacenamiento en la base de datos. A través de este modelo, se facilita la gestión y organización de las tareas, proporcionando una estructura uniforme para realizar operaciones como agregar, actualizar y consultar tareas de manera eficiente y consistente.

AUTORIZACIÓN REQUERIDA: ApiKey

Única por usuario. Permite crear, obtener y administrar tareas en un entorno de prueba para un usuario en específico.

Authorization type: bearer token
Bearer format: sk_test_xxxxxxxxxxx

REQUEST BODY SCHEMA: application/json

titulo string required
Representa el título o nombre de la tarea.
descripcion string required
Proporciona una descripción detallada de la tarea.
estado boolean
Indica si la tarea está completa o pendiente.
